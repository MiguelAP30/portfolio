# Foto de perfil

Deja aquí **un único archivo** con tu foto de perfil.

Formatos soportados: `jpg`, `jpeg`, `png`, `webp`, `avif`

Ejemplos de nombre válido:
```
foto.jpg
perfil.png
miguel.webp
```

El código la detecta automáticamente — no hay que tocar nada más.
Si no hay foto, se muestra un avatar con tus iniciales.
